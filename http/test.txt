1) Test 413 (Payload Too Large)

Port 8080 (limit 10M)
# create 11 MiB blob
truncate -s 11M big.bin  # or: dd if=/dev/zero of=big.bin bs=1M count=11

# try uploading to /upload/ (your POST handler)
curl -i -X POST --data-binary @big.bin http://127.0.0.1:8080/upload/big.bin

# If your server expects multipart:
curl -i -F "file=@big.bin" http://127.0.0.1:8080/upload/

/images/ allows GET only.

try DELETE
